    简单的预测方法¶

    均值法¶

    所有未来值的预测值等于历史数据的平均值。我们把历史数据记作\(y_{1},\dots,y_{T}\), 预测值就可以表示为：

    \[\hat{y}_{T+h|T} = \bar{y} = (y_{1}+\dots+y_{T})/T\]

    \(\hat{y}_{T+h|T}\)代表基于\(y_{1},\dots,y_{T}\)对\(y_{T+h}\)的估计值。

    Naïve 方法¶

    在 naïve 预测方法中，我们简单地将所有预测值设为最后一次的观测值，即：

    \(\hat{y}_{T+h|T} = y_{T}\)

    这种方法在很多经济和金融时间序列预测中表现得非常好。

    当数据服从随机游动过程( \(y_t = y_{t-1} + \varepsilon_t\) )时 Naïve 预测是最优的。

    趋势法¶

    相对于 Naïve 方法，趋势法（漂移法）允许预测值随着时间的推移增大或减小，并且我们假定单位时间改变量（称作 “漂移”, Drift）等于历史数据的平均改变量，因此
    \(T+h\)时刻的预测值可以表示为:

    \[\hat{y}_{T+h|T} = y_{T} + \frac{h}{T-1}\sum_{t=2}^T (y_{t}-y_{t-1}) = y_{T} + h \left( \frac{y_{T} -y_{1}}{T-1}\right)\]

    这相当于把第一个观测点和最后一个观测点连成一条直线并延伸到未来预测点。

    Python示例¶

    import matplotlib.pyplot as plt
import numpy as np


def make_average_prediction(y, h):
    avg = np.mean(y)
    return [avg] * h

def make_naive_prediction(y, h):
    return [y[-1]] * h

def make_drift_prediction(y,h):
    k = (y[-1] - y[0]) / (len(y) - 1)
    return [k*i + y[-1] for i in range(1, h+1)]

num_trn = 10
num_val = 3

def plot_predictions(predictions, label):
    plt.plot(range(num_trn, num_trn+num_val), predictions, label=label)
    plt.scatter(range(num_trn, num_trn+num_val), predictions)

y = np.linspace(0, 10, num_trn) + np.random.randn(num_trn)
plt.plot(range(num_trn), y)
plt.scatter(range(num_trn), y)

plot_predictions(make_average_prediction(y,num_val),'average')
plot_predictions(make_naive_prediction(y,num_val),'naive')
plot_predictions(make_drift_prediction(y,num_val),'drift')

plt.legend()

    季节性 Naïve 方法¶

    季节性 Naïve 方法与 Naïve 方法类似，它适用于季节性变化剧烈的数据。我们将每个预测值设为同一季节的前一期观测值（例如：去年的同一个月）。那么， \(T+h\) 时刻的预测值可以记作：

    \[\hat{y}_{T+h|T} = y_{T+h-km}\]

    等式中 \(m\)为周期长度，\(k\)是\((h-1)/m\)的整数部分
